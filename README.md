# AutoPrime 🚗

Sitio web estático de automotora desarrollado con React + Vite.

## Tecnologías utilizadas
- React
- Vite
- React Router DOM
- React Icons
- CSS Modules
- Google Fonts

## Funcionalidades
- Sitio responsive (desktop y mobile)
- Navegación con React Router DOM
- Cards de vehículos con dummy data
- Componente de propuestas con props
- Animaciones CSS puras
- Página de desarrollador
- Página 404
